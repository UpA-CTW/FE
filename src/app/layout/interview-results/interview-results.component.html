Question: Java code provides for class methods with both public and private access specifiers. What is the difference between these two modifiers? Answer: A public access specifier directs that the method be accessible both inside and outside of the class. This is true as long as the class, itself, is declared as public, or the calling class is within the same package. If the class does not specify a modifier then it is assumed to have default (package) access, and the declarations for its methods do not override this.
